# Modern To-Do List App

A simple, clean, and stylish To-Do List application built to help you manage your daily tasks. It features a modern Neumorphic (Soft UI) design.

# Features
- **Add Tasks:** Type your task and click the 'ADD' button.
- **Delete Tasks:** Click the trash icon to remove completed tasks.
- **Responsive Design:** Works perfectly on both Desktop and Mobile.

# Tech Stack
- **HTML5:** For the structure.
- **CSS3:** For the modern Neumorphic styling and layout.
- **JavaScript:** For functionality (Adding/Deleting tasks).

# Project Structure
- `index.html` - The main page.
- `style.css` - The design and colors.
- `script.js` - The logic and functionality.

# How to Use
1. Download or Clone the repository.
2. Open the `index.html` file in your browser.
3. Start adding your tasks!
